tizi: JSX component factory

The TypeScript compiler will parse JSX and translate it to JavaScript function calls, so to create a JSX factory in TypeScript all you have to do is implement a function and configure TS to invoke it (instead of for example React). tizi is a very small JSX factory with support for components. tizi just renders DOM. tizi components can expose an external API to manage UI updates.

Chill O'clock: geolocation-based approximate clock

What time is it? It's chill o'clock

Try it yourself

Chill O'clock breaks the day into 8 sections and the night into 4. The length of day and night is calculated from your latitude and longitude, and of course the date. Rather than giving a precise time it gives a rough idea of which part of the day or night you are in for your geographic location and the season - summer days are of course longer than winter days.
